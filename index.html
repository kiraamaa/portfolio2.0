About Me

Hi, I'm Kira. I’m a front end web developer and I believe in highly functional, streamlined apps that reflect modern design principles. I’m intrigued by the concept of pushing the boundaries of efficient and practical code, and I’m highly motivated by my desire to directly influence the future of design and development. Given my highly technical background and my passion for creative design, I’m looking for a design-oriented developer or product manager position at an established tech company so that I can help drive innovation with well-designed development solutions.

Here's what I excel at:

  • Writing HTML, CSS, and JavaScript for displays in modern browsers with responsive design practices.
  • Developing back-end web applications and APIs using a modern programming languages and MVC web development framework, such as Rails.
  • Collaborating with teams using an agile development workflow and widely-accepted collaboration practices using Git and GitHub for version control.
  • Improving and demonstrating web development skill-sets by building a series of full-stack web applications using JavaScript, jQuery, HTML5/CSS3, Bootstrap, and Ruby on Rails.
